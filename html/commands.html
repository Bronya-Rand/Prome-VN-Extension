Prome Commands

  1. /letterbox - Show/Hide the letterbox (black bars) in the VN UI.
  2. /sprite-focus - Toggles focus mode on character sprites.
  3. /set-focus-animation - Sets the focus mode animation.
  4. /sprite-defocus - Toggles the defocus tint on non-speaking character sprites.
  5. /sprite-shake - Toggles the shake animation when a character speaks on character sprites.
  6. /sprite-shadow - Toggles sprite shadows on character sprites.
  7. /tint - Toggles world/character tint on the VN UI.
  8. /tint-world - Toggles world tint on the VN UI.
  9. /tint-characters - Toggles character tint on the VN UI.
  10. /tint-share - Toggles sharing world tint with character sprites (This will override Character Tint).
  11. /express - Sets the expression of the user sprite.
  12. /set-user-sprite - Sets the user sprite set to use for the user sprite.
  13. /user-sprite - Toggles the user sprite on the VN UI.
