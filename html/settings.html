Features marked with a * require Prome to be enabled.

Sheld Configuration

Letterbox Configuration*
Select the letterbox mode for the Prome VN UI.
Select the color of the letterbox.
Set the size of the letterbox.
Sprite Configuration*
Set the maximum number of visible sprites that appears in the VN screen.
Sprite Shadow Configuration
Set the X offset of the character shadow.
Set the Y offset of the character shadow.
Set the blur of the character shadow.
Focus Mode Settings
Select the animation for focus mode.
Set the speed of the focus animation.
[BETA] User Sprite Configuration
Type the name of the sprite set to use for your persona. (Place your sprites in the 'characters' folder in SillyTavern).

Note: Create a sprite folder in the characters folder of your user directory (typically 'data/default-user'). Place your expressions there.

Tint Configuration
Select the tint preset to use for the Prome VN UI.
World Tint Settings Set the strength of the world blur. Set the brightness of the world. Set the contrast of the world. Makes the world black and white. Set the hue of the world. Inverts the world colors. Saturates the world colors. Makes the world warmer in color.
Character Tint Settings* Set the strength of the character blur. Set the brightness of the character. Set the contrast of the character. Makes the character black and white. Set the hue of the character. Inverts the character colors. Saturates the character colors. Makes the character warmer in color.
